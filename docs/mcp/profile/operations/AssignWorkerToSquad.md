# AssignWorkerToSquad
Assign a worker (survivor, defender or hero) to a squad by ID.

## Attributes
Compatible Profiles: `campaign`  
Supported MCPs: `fortnite`

## Payload
```json
{
    "squadId": "",
    "characterId": ""
}
```

## Parameters
- `squadId`: ID of a squad
- `characterId`: ID of a character
